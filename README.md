# Somali Election 2026

This is a simple web application built with **HTML**, **CSS**, and **JavaScript** to simulate voting for the Somali Election 2026. The app allows users to choose their preferred candidate and displays a confirmation message along with the candidate's image.

## Features

- Interactive voting buttons for 8 Somali presidential candidates:
  - Farmaajo
  - Shariif Sheikh Ahmed
  - Hassan Sheikh Mohamud
  - Abdirahman Abdishakuur (redirects vote to Hassan Sheikh Mohamud)
  - Saacid Shirdoon
  - Kheyre
  - Rooble
  - Dani
- Stylish UI with gradient background and blur effect.
- Smooth animation for vote confirmation.
- Candidate images displayed upon selection.

## How to Use

1. Open `index.html` in any modern web browser.
2. Click on your preferred candidate's button.
3. View your vote confirmation and candidate image.
4. Note: Voting for "Abdirahman Abdishakuur" will automatically count as a vote for "Hassan Sheikh Mohamud".
